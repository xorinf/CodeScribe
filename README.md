# AI-Enhanced Documentation Generator (CodeScribe)

CodeScribe is an intelligent system that automatically generates, updates, and maintains technical documentation for software codebases.

## Overview

It analyzes source code, version history, and team communications to produce accurate, up-to-date, and developer-friendly documentation without manual effort.

## Structure

- ``: Code parsing & AST generation
- `src/analyzer`: Semantic analysis of the codebase
- `src/generator`: NLP processing to generate documentation
- `src/integrations`: Git and CI/CD integrations
- `src/cli.py`: Command-line interface
